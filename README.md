# KNIME攻略ガイド（日本語）

KNIME を日本語で・視覚的に・体系的に学べる、初心者向けの非公式リファレンスサイトです。
ノードの解説、ループ処理、テンプレート、エラー解決を1ページにまとめています。

## 特徴

- **ノード辞典**：200種類のノードを日本語で解説（設定項目・入力例・Tips つき）
- **基本用語**：KNIMEの基礎用語100語を図解つきで解説
- **ループ解説**：代表的なループパターンを図解
- **テンプレート集**：よくある処理フローのひな形
- **エラー解決**：エラーメッセージから解決策を提案

依存ライブラリ・ビルド不要の**単一HTMLファイル**です。`index.html` をブラウザで開くだけで動作します。

## ローカルでの閲覧

```
# このリポジトリをクローンまたはダウンロードして
open index.html        # macOS
# もしくは index.html をブラウザにドラッグ&ドロップ
```

## 公開（GitHub Pages）

`main` ブランチの `index.html` がそのまま公開ページになります。
リポジトリの **Settings → Pages** で公開設定を行ってください。
